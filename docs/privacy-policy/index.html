Pulse Dock Privacy Policy

Effective date: June 27, 2026

Pulse Dock is a local macOS system monitor. It samples on-device system metrics so it can display the dashboard, menu bar popover, and WidgetKit extension.

Data We Process On Device

Pulse Dock uses public macOS APIs to read local system status, including CPU activity, memory pressure, storage capacity, network counters, battery and thermal state, display information, GPU capability, uptime, and running app status.

The app stores settings, sanitized trend history, and a compact latest widget snapshot locally on your Mac. The widget snapshot is shared with the WidgetKit extension through App Group UserDefaults so the widget can display recent local metrics.

Data Collection

Pulse Dock does not collect personal data from this app. The current app does not:

  • Create user accounts or require sign-in.
  • Track users across apps or websites.
  • Send analytics, advertising identifiers, or remote telemetry to the developer.
  • Upload system metrics, running app lists, screenshots, documents, or personal files.
  • Include third-party advertising or analytics SDKs.

Data Sharing

Pulse Dock does not sell, rent, or share user data with third parties. Local app data stays on the Mac unless you choose to share information when contacting support.

Retention And Removal

Local settings, trend history, and widget snapshots remain on your Mac for app functionality. You can remove local app data by deleting the app and its associated local containers from macOS.

Children's Privacy

Pulse Dock is a general utility app and is not directed to children. Because the app does not collect personal data, it does not knowingly collect personal information from children.

Changes To This Policy

If Pulse Dock adds accounts, analytics, crash reporting, remote services, or any other data collection feature in the future, this policy and the App Store privacy answers will be updated before release.

Contact

For support or privacy questions, use the Pulse Dock support page.
