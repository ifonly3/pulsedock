# Pulse Dock Privacy Policy

Effective date: 2026-06-19

Pulse Dock is a local macOS system monitor. It samples on-device system metrics such as CPU, memory, storage, network counters, battery, thermal state, displays, GPU capability, uptime, and running app status using public macOS APIs.

Pulse Dock does not create accounts, collect personal data, track users, run analytics, send crash reports, or perform remote network probes.

## Data Stored On Device

Pulse Dock stores app settings, sanitized trend history, and a compact latest widget snapshot locally on the Mac. The widget snapshot is shared with the WidgetKit extension through App Group UserDefaults. This local data is used only to display the dashboard, menu bar popover, and WidgetKit extension.

## Data Sharing

Pulse Dock does not transmit user data to the developer or to third parties.
