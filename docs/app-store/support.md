# Pulse Dock Support

Public page source: `docs/support/index.html`

Pulse Dock is a local macOS system monitor for the main app, menu bar popover, and WidgetKit extension.

## Contact

For support, questions, bug reports, or App Review follow-up, open an issue in the public repository:

https://github.com/ifonly3/pulsedock/issues

Please do not include private data, personal files, screenshots containing sensitive information, or confidential system details in public issues.

## What To Include

When reporting a problem, include:

- macOS version
- Pulse Dock version and build number
- Whether the issue appears in the main window, menu bar popover, or widget
- A brief description of what happened and what you expected
- Steps that reproduce the issue, if you have them

## Supported Platform

Pulse Dock supports macOS 14.0 or later. Release builds are prepared for both Apple silicon and Intel Mac hardware unless a specific release note says otherwise.

## Privacy

Pulse Dock is designed as a local utility with no account, no tracking, and no analytics. Read the privacy policy:

https://ifonly3.github.io/pulsedock/privacy-policy/
